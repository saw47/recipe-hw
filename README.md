## Нетология, итоговая приложенька
### Главный экран
* строка поиска - слушает ввод, сразу фильтрует
* крестик в строке - очищает строку, скрывает клавиатуру
* фильтр - перезод на фрагмент фильтра - чекбокс по типу рецепта
* плюс - переход на фрагмент добавления нового рецепта
* тогл все-избранное - фильтр по избранному
* контейнер для рецептов - при пустом списке фотка, если есть объекты - фон
* клик по карточке - переход на расширенную карточку
* лонгтап по карточке - перемещение - появятся кнопки вверх/вниз. Снова клик по карточке - кнопки исчезнут, перемещение выключится.

### Главный экран - карточка рецепта
* Название рецепта
* категория рецепта
* справа от них кнопка меню - там редактировать и удалить
* снизу поле - автор рецепта
* над ним кнопка "добавить в избранное", при клике красится

### Расширенная карточка рецепта
* Название рецепта
* категория рецепта
* справа от них кнопка меню - там добавить шаг, редактировать и удалить
* под ними поле - автор рецепта
* Контейнер с шагами
* внизу справа кнопка "добавить в избранное", при клике красится
* лонгтап по карточке шага - перемещение - появятся кнопки вверх/вниз. Снова клик по карточке - кнопки исчезнут, перемещение выключится.

### Карточка редактирования рецепта
* меню в топбаре - крестик - выход без сохранения
* оно же - дискета - сохранение
* автор рецепта
* Название рецепта
* категория рецепта - спиннер
* Контейнер с шагами
* внизу по центру кнопка "добавить шаг" - переход на фрагмент Добавить шаг
* лонгтап по карточке шага - перемещение - появятся кнопки вверх/вниз. Снова клик по карточке - кнопки исчезнут, перемещение выключится.

### Карточка шага
* Текст шага
* кнопки редактировать и удалить
* картинка - заглушка, изменение не реализовано

### Добавить шаг
* меню в топбаре - крестик - выход без сохранения
* оно же - дискета - сохранение
* поле описание - текст про шаг
* снизу фото, на нем кнопка удалить - загрузка фото не реализована, кнопка кликается тостом
